# Programmation1-UPEC

Ce dépôt contient les travaux dirigés réalisés en L1 Informatique à l'UPEC, utilisant un dérivé de Java, nommé [Robusta](https://github.com/FrenchTechLead/robusta).

## Installation

Pour installer Robusta, vous pouvez suivre les instructions sur le [dépôt GitHub officiel de Robusta](https://github.com/FrenchTechLead/robusta)
Une utilisation de **Visual Studio Code** est recommandée.

## Utilisation

Une fois Robusta installé, vous pouvez compiler et exécuter vos programmes en utilisant les commandes suivantes :

```bash
$ java -jar robusta.jar compile fichier.jvs
$ java -jar fichier.jar
```
Pour plus de facilité, tous les fichiers sont déjà compilé. L'installation est Robusta est alors facultative pour essayer les programmes.
A noter que le dépôt contient tous les Travaux sur Machine ``TM``, ainsi que quelques exemples vu en cours magistral ``CM``

## Documentation

La documentation de Robusta est très limitée, mais vous pouvez trouver quelques informations utiles sur le wiki du dépôt GitHub : [Wiki robusta](https://github.com/FrenchTechLead/robusta/wiki)
